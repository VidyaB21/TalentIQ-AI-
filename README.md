# TalentIQ-AI

## Enterprise AI-Powered Resume Intelligence Platform

TalentIQ-AI is an enterprise-grade Resume Intelligence Platform designed to automate resume analysis, candidate evaluation, skill-gap identification, and career recommendation workflows.

The platform leverages Artificial Intelligence, Natural Language Processing (NLP), and a scalable microservices architecture to transform unstructured resumes into actionable insights. It enables intelligent candidate assessment through resume parsing, ATS scoring, skill-gap analysis, and career path prediction.

Built using React, ASP.NET Core, FastAPI, MongoDB, and Docker, the platform demonstrates modern software engineering practices, API-driven architecture, containerized deployment, and AI-powered decision support systems.

---

## Key Features

### AI Resume Parsing

* Extracts candidate information from uploaded resumes
* Identifies skills, education, certifications, projects, and work experience
* Converts unstructured resume content into structured candidate profiles

### ATS Resume Scoring

* Evaluates resume quality and ATS compatibility
* Generates performance scores based on industry-relevant criteria
* Highlights optimization opportunities for improved visibility

### Skill Gap Analysis

* Compares candidate skills against target job requirements
* Detects missing competencies and knowledge gaps
* Provides actionable upskilling recommendations

### Career Path Prediction

* Analyzes candidate profiles and skill sets
* Recommends suitable career opportunities and job roles
* Supports data-driven career planning and professional growth

### Candidate Intelligence Dashboard

* Centralized view of candidate insights
* Resume analytics and performance metrics
* Streamlined evaluation workflow for recruiters and hiring teams
